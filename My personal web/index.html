HTML(Hyper Text Markup Language) is a programming body for building website

*In html there are types of tags namely
1.opening tags
2.closing tags
3.self-closing tags
*HTML elements include
1.Headings:(h1,h2,h3......h6)
2.paragraphs:(p)
3.links:(a)
4.images:(img)
*Some HTML attribute include
1.id:unique identifier
2.class:applies styles or behaviors
3.style:applies in line styles
4.href:specifies link destination

In order to create a link to open another website you must type this in your editor:
1. href="https://www.example.com">
with the [a tag openning and closing]

*Some additional attribute
1._blank:opens the link in a new tab or window
2._self:opens the link in the current tab
3.title:adds a toolpit with additional information about the link
